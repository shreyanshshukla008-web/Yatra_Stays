# PROJECT_MERN

A React + Vite hotel listing app.

## Features
- Browse a list of hotels
- View hotel details
- Search and filter experiences

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/AeriX00/PROJECT_MERN.git
   cd PROJECT_MERN
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```

The app will open on http://localhost:5173 by default.

## Build

```bash
npm run build
```
